앱 추가하는 방법

앱은 구글 스프레드시트에서 관리합니다.
시트에 새 행을 추가하면 자동으로 이 페이지에 반영됩니다.

  1. 구글 스프레드시트 > 앱목록 시트 열기
  2. 새 행에 앱이름 | 설명 | URL | 카테고리 | 등록일 입력
  3. 카테고리: 퀴즈 / 시뮬레이션 / 게임 / 도구 중 선택
  4. 저장하면 다음 방문 시 자동 반영
